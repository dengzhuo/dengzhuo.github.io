CentOS/Ubuntu一键安装IPSEC/IKEV2 VPN服务器

发表于 2017-11-12 | 分类于 VPN

详细内容查看：

https://quericy.me/blog/699/

包括搭建方法

流程：

先安装加速器
参照

https://www.xiaoz.me/archives/7973

linode服务器不支持安装内核，需要再后台配置内核

再安装VPN

安装apache：
yum install httpd Cy

配置apache登录用户：
htpasswd -c -b /etc/httpd/htpasdb zhanghao mima

在apache配置文件/etc/httpd/conf/httpd.conf最后追加：


options indexes
AuthType Basic
AuthName “!!!”
AuthBasicProvider file
AuthUserFile /etc/httpd/htpasdb
Require user zhanghao

拷贝客户端证书：
mkdir /var/www/html/key
cp -p ca.cert.pem /var/www/html/key/
cp -p client.cert.p12 /var/www/html/key/
重启服务，service httpd restart
然后就可以在客户端访问：http://****/key 下载证书。
